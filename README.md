# Arc Beer Store

Arc Beer Store là một ứng dụng phi tập trung (dApp) chuyên nghiệp, cho phép người dùng trải nghiệm mua bia trực tuyến an toàn và minh bạch với thanh toán bằng USDC trên mạng lưới Arc.

## 🌟 Tính năng nổi bật
- **Giao diện hiện đại:** Thiết kế tối giản, trực quan với Tailwind CSS và Framer Motion.
- **Thanh toán Web3:** Tích hợp trực tiếp với ví thông qua Wagmi/Viem.
- **Thanh toán nhanh:** Hỗ trợ quét mã QR để chuyển USDC thanh toán tức thì.
- **Minh bạch:** Mọi giao dịch được xử lý trực tiếp trên blockchain thông qua `BeerManager` smart contract.

## 🛠 Công nghệ sử dụng
- **Frontend:** React, TypeScript, Vite, Tailwind CSS.
- **Web3:** Wagmi, Viem.
- **Smart Contracts:** Solidity (OpenZeppelin).
- **Deployment:** Vercel.

## 🚀 Hướng dẫn phát triển
1. **Clone dự án:**
   ```bash
   git clone https://github.com/Batinho13/arc-beer-store.git
   cd arc-beer-store
   ```
2. **Cài đặt dependencies:**
   ```bash
   cd frontend
   npm install
   ```
3. **Chạy ứng dụng:**
   ```bash
   npm run dev
   ```
